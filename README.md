# javascript-modal
Javascript modal - tiny and simple.

## Usage example
### Script
```javascript
var modal = new Modal;
```

### View
```html
<button class="modalBtn" data-target="#modal-default">Default</button>

<div class="modal-container" id="modal-default">
    <div class="modal-wrapper">
        <i class="modal-close"></i>
        ...
    </div>
</div>
```
